1. 이미지,영상 전용 서버로 많이 사용
2. 백업 서버로 많이 사용
3. 웹 호스팅(빌린다) 서버로 사용(v) -> 정적 웹 
>+ =>S3임대해서 내 HTML을 S3에 업로드 하겠다.

- 정적 웹: HTMl , JS , JQuery
- 동적 웹: (웹 프레임워크 + 템플릿엔진)
