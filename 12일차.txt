4.문법
*함수 이름은 동사
 <접근지정자> <리턴타입> <함수 이름> <인자값>
 *인자값에는 최대 3개까지만 오자.
 (1.인자값 2.파라미터값)
  public int getSum(int x, int y, int z){
    int sum = 10;
    //리턴타입이 int형이면
    //int형 데이터를 반환(return)해야 함.
    return sum;   
  }  

  <접근지정자> <리턴타입> <메소드이름> <인수값>
  1.리턴타입
  메소드의 최종 결과값을 반환.
  리턴 타입에는 데이터타입(int,doble,float,long...) 8개와 클래스타입(String) 1개
  총9개.
  리턴 타입이 없을떄는 void로 표현

  2.인자값 (=파라미터값)
  데이터타입(int.doble,float,long...)8개와 클래스타입 1개(String)총9개가 위치
  인자값(파라미터값)이 없을 떄는 ()반 괄호
  *** 인자값(파라미터값)은 3개 이하로 작성.

  3.접근지정자
   1.public   :    다른 클래스에서 호출 가능
   2.private  :    내부 클래스에서만 호출 가능
   3.protected
   4.default
